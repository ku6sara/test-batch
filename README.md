yt-dlp.exeとffmpeg一式必須です
https://github.com/yt-dlp/yt-dlp#installation
↑のRELEASE FILESのRecommendedからwindows用のyt-dlp.exeを落とす

https://www.ffmpeg.org/download.html
↑からwindowsの窓からWindows builds from gyan.devをクリック　ffmpeg-git-essentials.7zを落として解凍、binフォルダをコピー

完成図
C:\YT-DLP
│
├─ここに.batファイル！！
├─assets　←ここに落とした動画だけでてくる！！！！
└─bin
    │  ffmpeg.exe
    │  ffplay.exe
    │  ffprobe.exe
    │  yt-dlp.exe
    └─temp
       └─ここに解凍したいらんファイル入れてる正直いらん
